📋 AutoPXE Ubuntu 极简使用说明

✅ 必须做的 3 件事
文件就位：将脚本 autopxe_ubuntu.sh 和 Ubuntu ISO 镜像放在同一个文件夹内。
网络隔离：确保运行环境没有其他 DHCP 服务器（如拔掉路由器 LAN 线，仅用交换机或网线直连目标电脑），防止 IP 冲突。
执行脚本：在终端进入该目录，运行：
      sudo ./autopxe_ubuntu.sh
   

⚡ 简单说明
自动识别：脚本会自动发现 ISO，识别是 Desktop 还是 Server 版，无需手动配置。
自动网络：脚本会自动读取本机 IP 和网关，动态设置 DHCP 范围，适配任何网段。
启动安装：脚本运行显示“就绪”后，将目标电脑设为网卡启动 (PXE) 即可自动进入安装界面。
退出清理：按 Ctrl + C 停止脚本，系统会自动清理所有临时配置，恢复原状。
